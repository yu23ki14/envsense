# 部品実測寸法メモ（Issue #1 / Phase 0）

XIAO ESP32S3 Sense + OV2640 カメラ + LiPo バッテリーの実測寸法。
Phase 1（#2）の OpenSCAD パラメトリック筐体設計の基準値。

## 測定ルール

- 単位は **mm**。デジタルノギス 0.05mm 単位、各寸法 3 回測って中央値を記録。
- **実測値は「素のサイズ」を記録する。** はめ合いのクリアランスは OpenSCAD 側で足す。
- **マスター原点 = XIAO メイン基板の USB-C 側短辺の角。** 主基板の部品はこの原点基準。
- 拡張ボード／カメラの X/Y は各ローカル原点基準。実空間へは `assy_offset` で変換。
- 上面 = USB-C 側 / 下面 = Sense 拡張ボードが付く側。

### 静電タッチ長押し（電源 ON/OFF も兼ねる）

- ESP32S3 は GPIO1〜14 が容量タッチ対応。
- 役割: 長押しでディープスリープ ⇄ 復帰（＝電源 OFF/ON）。意図的キャプチャ等の操作もここに集約。
- タッチパッドに繋ぐピン: 未定（カメラ/マイク/SD で GPIO 逼迫、firmware と要調整）。タッチ起床は RTC GPIO 対応ピンが必要。
- タッチ面（壁厚を ~1.5〜2mm に薄くする面）: **カメラ側の面**。

### 設計メモ: 電源 ON/OFF（決定済み）

- **方式: 静電タッチ長押し → ディープスリープ。物理スイッチは設けない。**
- バッテリーは常時 `BAT` パッドに接続されるため、充電は電源状態によらず常に可能。
- ディープスリープ中の ESP32-S3 は数十 µA 程度で、実用上「OFF」とみなせる。
- 筐体に電源スイッチ用の開口が不要 → #30 防水で有利。

### 設計メモ: USB-C 開口

- USB ケーブルのプラグ樹脂モールドはケーブルごとに異なるため、特定寸法は測らない。
- 開口は受け口（8.3×2.5）より広め、一般的な USB-C プラグが挿せる ~12〜13mm 幅で設計し、USB 端の壁は薄く（または面取り）する。
